== Description ==

Allow customers to pre-order from your store.

 
## Features
 
- Allow pre-purchases on your website
- Works with simple product only
- Offer your customers a clear and updated overview of their pre-orders
- Use the automatic email system for admins and customers
- Encourage customers to purchase with customized text
- Offer your customers a clear and updated overview of their pre-orders
- Have a definite and updated pre-orders overview
 
== Installation ==

    = Installation from within WordPress =

        1. Visit 'Plugins > 'Add New'.
        2. Search for 'Pre-Order'.
        3. Install and activate the 'Pre-Order' plugin.

    = Manual installation =

        1. Upload the entire `Pre-Order` folder to the `/wp-content/plugins/` directory.
        2. Visit 'Plugins'.
        3. Activate the `Pre-Order` plugin.
